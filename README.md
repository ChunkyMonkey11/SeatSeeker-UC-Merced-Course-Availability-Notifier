# 🎓 SeatSeeker

**SeatSeeker** is a Python-based email notification tool that monitors UC Merced’s course registration page and notifies you as soon as a section becomes available. Never miss your chance to enroll in the class you need again.

---

## 🚀 Features

- ✅ Real-time course availability checker
- 📧 Email alerts via Gmail SMTP
- 🔍 Debug mode for HTML inspection and testing
- 🧪 Easy to test with a simulated “open” section trigger

---

## 🛠️ Technologies Used

- Python 3
- [Requests](https://pypi.org/project/requests/) – for HTTP requests
- [BeautifulSoup](https://pypi.org/project/beautifulsoup4/) – for HTML parsing
- [smtplib](https://docs.python.org/3/library/smtplib.html) – for sending emails
- `email.mime` – for email formatting
